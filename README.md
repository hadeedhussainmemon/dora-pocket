## üöÄ Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/dora-pocket.git
    cd dora-pocket
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the app**:
    Visit [http://localhost:3000](http://localhost:3000) to see the magic!
